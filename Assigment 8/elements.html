HTML Elements Showcase – Inline vs Block

In HTML, elements are broadly divided into two categories: block-level and inline. Block elements take up the full width and start on a new line, while inline elements only take as much space as needed.

Block Elements | Inline Elements | Comparison

Block Elements

1. <div>

This is inside a div container.

The <div> is a block-level container often used for grouping.

2. <p>

This is a paragraph of text.

The <p> tag represents a block of text, always starting on a new line.

3. <h1>–<h6>

Heading 1

Headings are block elements that define document structure.

4. <ul> and <ol>

  • Apple
  • Banana

  2. Second

Unordered lists use <ul> and ordered lists use <ol>.

5. <blockquote>

"Coding is the new literacy."

The <blockquote> tag is for long quotations.

6. <hr>

Above is a horizontal line:

The <hr> tag represents a thematic break.

7. <table>

The <table> tag is a block element for tabular data.

8. <section>

This is a section

Sections group related content.

9. <article>

News Article

This is an article block element.

10. <nav>

Navigation menu goes here.

The <nav> tag defines a navigation block.

Inline Elements

1. <span>

This is inside a span.

Span is an inline container with no line break.

2. <a>

Visit Google now.

The <a> tag creates hyperlinks.

3. <b> and <strong>

This is bold and this is important.

Both show bold, but <strong> has semantic meaning.

4. <i> and <em>

This is italic and this is emphasized.

Both slant text, but <em> adds importance.

5. <u>

This is underlined text.

The <u> tag underlines inline text.

6. <sup> and <sub>

E = mc2 and H2O.

Superscript (<sup>) goes above; subscript (<sub>) goes below.

7. <mark>

This is highlighted text.

The <mark> tag highlights text.

8. <abbr>

HTML stands for HTML.

The <abbr> tag defines abbreviations.

9. <code>

Use console.log("Hello") in JavaScript.

The <code> tag represents programming code.

10. Mixed inline example

You can combine, mix, and highlight inline tags.

Inline elements work inside sentences.
